About Machine Lending

Introduction

Peer-to-peer lending is a relatively new form of financing, from the people to the people. One of the first, and currently largest, such lending platforms is Lending Club. Click here for more information about Lending Club and how democratized lending works.

Lending Club and other platforms generally have an "Auto-Invest" option to allow investors to minimally manage their investment. Lending Club, for instance, allows one to choose a distribution of loan grades in which to invest. However, choosing a distribution to optimize returns is not trivial or obvious. This is the reason Machine Lending was developed.

The term Machine Lending is ment to describe the fact that the loan default predictions on this site are calculated using state-of-the-art machine learning algorithms. The model used to make the predictions is constantly evolving and improving. See the Methods for more details.

Data

As any good and transparent corporation would do, Lending Club has made a large amount of historical data available. This data ranges from interest rate and FICO score to the number of months since the borrower was delinquent and their revolving balance. Crucially, it also includes the current state of the loan. This means one can see if a loan was paid in full, charged off, or current. That information is used as the dependent variable when creating the Lending Club model. Everything else is used to train the models to predict loan default rate.

In addition to historical data, Lending Club also provides information about the current loans that require funding via their API. This enables algorithmic prediction and purchase of the cream of the crop loans.

Methods

To make loan default predictions, Machine Lending uses what is called ensemble learning. In short, take a set of the appropriate machine learning algorithms and optimize their coordination to fit the historical data as best possible. It's a "wisdom of crowds" approach to a democratized lending platform. Another way to think of it is meta-classification. The source code for the in-house meta classifier is available on github
